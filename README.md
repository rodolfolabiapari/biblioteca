# Minha Biblioteca de Livros

## O que é?

Minha base de dados, levemente confiável, dos livros que tenho.

Até o momento, contempla somente o que tenho na estante, ou seja, livros
físicos.

## Como foi populado?

A base foi construida usando aplicativo para android `Binary Eye` e/ou `Libib`.
Escaneei todos os ISBN de todos os livros e depois importei no Calibre.

Os metadados do Calibre, capas e afins não são confiáveis.

termino.
